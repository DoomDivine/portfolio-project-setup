# Research Methodology

## Tools Used

- Cursor IDE
- Claude Code
- Codex
- GitHub

## Collection Process

1. Identified 10 high-signal B2B SaaS LinkedIn creators.
2. Used Claude Code and Codex to help locate relevant content sources.
3. Reviewed LinkedIn posts manually.
4. Reviewed YouTube videos and transcripts.
5. Extracted key insights and summarized them.
6. Organized findings into structured research files.
